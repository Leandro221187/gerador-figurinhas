# Gerador de Figurinhas

Um aplicativo para gerar figurinhas personalizadas.

## Sobre

Gerador de Figurinhas é um projeto que permite criar e customizar figurinhas de forma fácil e intuitiva.

## Funcionalidades

- ✨ Criar figurinhas personalizadas
- 🎨 Customizar cores e estilos
- 💾 Salvar figurinhas criadas
- 📤 Compartilhar figurinhas

## Requisitos

- Node.js 16.0 ou superior
- npm ou yarn

## Instalação

```bash
git clone https://github.com/Leandro221187/gerador-figurinhas.git
cd gerador-figurinhas
npm install
```

## Como Usar

```bash
npm start
```

O aplicativo estará disponível em `http://localhost:3000`
